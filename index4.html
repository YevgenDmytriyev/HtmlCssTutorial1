What is the web server refers to both software and hardware
  1. On hardware side a web server is a computer that stores when server software and websites component files for eg. HTML, images, CSS stylessheets, javascript files it connects to internet and supports physical data interchange with other devices connected to the web
  2. On software side, a web server includes parts that control how web users access hosted files, This is HTTP server. An HTTP server is software that understands URLs (Web addresses) and HTTP(protocol which your web browser uses)
  • 1. once it gets request, HTTP server checks if the requested URL matches the existing file
  • 2. If so , the web server sends the file content back to the browser. If not, the server will check if should generate a file dynamically for the requests
  • 3. if neither of those options are possible, then web server returns error message to the browser. Ussualy its 404 error page
  • A web browser sends an HTTP request in the form of a GET parameter to the web server and asks it for an HTML document, for example, web page content or a file. The web server transmits the requested document to the web client in its HTTP response.
